# Datos Públicos - Trabajo de Título
###### (TextMining realizado en el Trabajo de Título: "Text Mining Aplicado a Documentación de API para la Detección de Directivas" - 2015)

Este repositorio contiene los archivos principales que fueron utilizados para realizar Text Mining en Weka o usados para realizar revisiones manuales de los comentarios de API en búsqueda de directivas.

## Contenido
- datos finales/:
  - Datos finales utilizados en los experimentos del trabajo.
    - dataset---apache---ordered.csv
    - dataset---jface---ordered.csv
    - dataset---java---ordered.csv
  - Registro de las clases y métodos contenidos en los datos finales:
    - clases_apache.txt
    - clases_java.txt
    - clases_jface.txt
    - metodos_apache.txt
    - metodos_java.txt
    - metodos_jface.txt
  - GatheredData-datosFinales.xlsx (archivo utilizado para ir anotando los datos obtenidos)

- datos preliminares/:
  - Datos finales utilizados en los experimentos del trabajo
    - DATASET-datosPreliminares.csv
  - Registro de las clases, métodos y miembros de clase contenidos en los datos finales:
    - clases_datosPreliminares.txt
    - metodosYMiembros_datosPreliminares.txt
  - GatheredData-datosPreliminares.xlsx (archivo utilizado para ir anotando los datos obtenidos)
